# Gestionnaire de Tâches

Application web moderne de gestion de tâches développée avec HTML5, CSS3 et JavaScript ES6 Modules.


## Fonctionnalités

- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Compléter une tâche
- Recherche en temps réel
- Filtres de tâches
- Validation du formulaire
- Responsive design
- Sauvegarde avec localStorage



## Technologies utilisées

- HTML5
- CSS3
- JavaScript ES6
- LocalStorage
